<h1>🍔 Landing Page de Restaurante</h1>

<p>Landing page moderna e responsiva desenvolvida para um restaurante, com foco em apresentação visual atrativa, experiência do usuário e navegação intuitiva.

O projeto simula a presença online de um restaurante, destacando seus principais produtos, categorias e identidade visual.</p>

<br>

<h2>🚀 Funcionalidades</h2>
<ul>
<li>Layout totalmente responsivo (mobile, tablet e desktop)</li>
<li>Seção de destaque com produtos principais</li>
<li>Menu interativo com categorias</li>
<li>Design moderno e atrativo</li>
<li>Navegação fluida entre seções</li>
<li>Botões de ação (call-to-action)</li>
</ul>

<h2>🛠️ Tecnologias utilizadas</h2>
<ul><li>HTML5</li>
<li>CSS3</li>
<li>JavaScript</li>
</ul> 

<h2>📸 Preview</h2>
